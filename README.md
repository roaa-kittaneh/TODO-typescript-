
## Projects Included

### 1. Counter App 
A simple counter application to understand state management and event handling.
* **Features:** Increment, Decrement, and Reset functionality.
* **Concepts Used:** `useState` hook, Event handling (`onClick`).
* **Testing:** Includes unit tests to verify the initial state (starts at 0) and the increment button functionality.

### 2. To-Do List 
A functional task management app demonstrating component hierarchy and data flow.
* **Features:** Add new tasks, toggle completion status, and delete tasks.
* **Concepts Used:** * Component composition (`<TodoList>` parent → `<TodoItem>` child).
  * Props drilling (passing data and functions to children).
  * Conditional rendering & inline styles (strikethrough for completed tasks).
  * Event handling (`onChange`, `onClick`).

## Technologies Used 
* **React** 
* **TypeScript** 
* **Vite** (Build Tool)
* **Vitest & React Testing Library** (Unit Testing Environment)

## How to Run 
1. Run `npm install` to install dependencies.
2. Run `npm run dev` to start the local development server.
3. Open the provided localhost link in your browser.

## How to Run Tests 
1. Run `npm run test`.
